# Smart-study-planner-
it is a web application that helps students to organize and manage their academic task 
What the project does

I developed the Smart Study Planner to help students organize their academic tasks, set deadlines, track progress, and manage their study schedule more effectively.

Why the project is useful

I created this project to help students avoid missed deadlines, stay organized, improve time management, and reduce academic stress.

How users can get started

To use the project, open the project folder in Visual Studio Code and run the index.html file using Live Server or any modern web browser. Users can then start adding and managing their study tasks.

Where users can get help

If users need help, they can read the project documentation or contact me for assistance.

Who maintains and contributes to the project

I developed and maintain this project as part of my APT3065 coursework. At the moment, it is an individual project, but it can be improved with additional features in the future.
